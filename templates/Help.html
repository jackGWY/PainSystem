About PainDatabase

PainDatabase is a database system for pain-related drug infomation researchers. This database integrates some of the pain-related information of the drug,target,pathways,diseases,sideeffects. The basic data for drugs and target proteins are derived from DrugBank,the basic data of sideeffect from SIDER, and the basic data for the pathway and disease is from KEGG.


How to Search

PainDatabase supply the searching function, you can enter the keywords in the input field and click search button on navigation bar. Now, the website supplies drug name,drug Accession Number, CAS number,sideeffect name,target name,target entry, disease name,KEGG Entry,and pathway name fuzzy searching.
